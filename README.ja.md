# comical

福井県を拠点とする若者コミュニティ「Comical Youth Fukui」(@comicalyouth)のメンバーを知るための楽しいBINGOゲームです。

## デモ

[ComicalメンバーBINGOをプレイ](https://code4fukui.github.io/comical/)

画面上部に5x5のBINGOカード、下部にコミュニティメンバーのリストが表示されます。ボタンをクリックしてメンバーに「会う」と、カード上の写真がフルカラーで公開されます。

## 遊び方

1. メンバーの写真が配置された独自の5x5のBINGOカードが生成されます。中央のマスはフリースペースです。
2. 最初はすべてのメンバーのマスが隠され、グレースケールで表示されます。
3. メンバーリストまでスクロールします。誰かに会ったら、その名前の下にある「会った！」ボタンをクリックしてください。
4. BINGOカード上の該当するマスがフルカラーで表示されます。
5. 進捗はブラウザに自動保存されます。ページ下部のリセットボタンを押すと、新しいゲームを開始できます。

## データソース

メンバーのリストと写真は、以下のオープンデータファイルから取得しています:
- [Comicalメンバーオープンデータ (CSV)](https://code4fukui.github.io/comical/member.csv)

コミュニティの詳細については、Instagramページをご覧ください:
- [Instagramの@comicalyouth](https://www.instagram.com/comicalyouth/)
