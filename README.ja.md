# こみかる

こみかるは、福井市を中心に活動するユース団体です。こみかるメンバーのBINGOゲームと、メンバーのオープンデータを提供しています。

## デモ
- [こみかるメンバーBINGO](https://code4fukui.github.io/comical/)
- [こみかるメンバーオープンデータ(CSV)](https://code4fukui.github.io/comical/member.csv)

## 機能
- こみかるメンバーのBINGOゲーム
- こみかるメンバーのオープンデータ
