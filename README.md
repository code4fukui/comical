# comical

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A BINGO game about the members of "Comical Youth Fukui" (@comicalyouth), an Instagram community in Fukui, Japan.

## Demo
[Comical Member BINGO](https://code4fukui.github.io/comical/)

## Features
- BINGO card with photos of Comical Youth Fukui members
- Click on member photos to "meet" them
- Card is saved in browser storage and can be reset

## Data
- [Comical Member Open Data (CSV)](https://code4fukui.github.io/comical/member.csv)
- [@comicalyouth on Instagram](https://www.instagram.com/comicalyouth/)
